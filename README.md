# ally-s-adventures

## 🌸Installation and Set up
1. Clone this repository
   ```
   git clone https://github.com/Sariah-dotcom/allys-adventures.git
   ```
2. Install python and pygame
   ```
   pip install pygame
   ```
3. Run the game
   ```
   python main.py
   ```
